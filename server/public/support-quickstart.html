Quickstart Guide

Go from zero to a running server in five steps.

  1. Pick a product line. Browse Cloud Compute, Telecom, or Storage from Services.
  2. Choose a region and workload. Filter by US-East, EU-Central, or APAC depending on where your traffic lives.
  3. Configure hardware. Use the Server Configuration Wizard to set CPU, memory, networking, and security options.
  4. Compare pricing. Check the Pricing page to see how your configuration maps to Starter, Business, or Enterprise plans.
  5. Add to cart and check out. Review your Cart and confirm.

Stuck partway through? Contact support or ask the 🧭 assistant in the corner — it can walk the site and do these steps for you.
